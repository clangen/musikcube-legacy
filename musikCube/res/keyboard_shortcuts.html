Keyboard shortcuts

  • Action-related shortcuts use ctrl as modifier

  • View-related shortcuts use alt

  • Use additionally shift when running out of meaningful 2-key combinations

  • Only use alphanumeric keys since any other character such as ] or ' makes it messy/non-standard with international layouts.
